Summary: Web accessibility means that people with disabilities can use the web equally.

What is website accessibility?

The web is for all people and is a important part to our daily lives. Websites that are properly designed and coded means that people with disabilities can use them. Some websites are not developed correctly meaning that these accessibility issues make the websites difficult to use. This is a great video with a thorough break down on what web accessibility is from the people at W3C. Web accessibility has many advantages.
